## Run 
Open Smudge.py and specify
1) PATH_TO_ORIGIAL_IMAGES : Directory path to the original images  
2) PATH_TO_DEST : Destination Directory, where the Augmented images will be stored 

Then simply run
```
python Smudge.py

```
